# Industrial Maintenance Manager

Java-based backend system for industrial equipment management.

## Overview
This project simulates a real industrial maintenance environment, allowing equipment registration and management using a layered architecture.

## Features
- Equipment CRUD (Create, Read, Delete)
- Layered architecture (Model, Service, Repository)
- Basic validation rules
- In-memory data storage

## Architecture
- Model: Represents domain entities
- Service: Business logic and validation
- Repository: Data storage layer

## Technologies
- Java
- Maven
